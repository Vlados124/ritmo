☁️ Данные и синхронизация
Отсканируй с другого устройства для импорта
1
Скачай ritmo-backup.json (кнопка «Экспорт» выше) — он понадобится позже
2
Зайди на github.com → создай бесплатный аккаунт → нажми New repository
3
Назови репозиторий ritmo, выбери Public, нажми Create
4
Нажми Add file → Upload files и загрузи файл planner.html
5
Зайди в Settings → Pages → Branch: main → Save
6
Через минуту ссылка будет: username.github.io/ritmo/planner.html — делись с друзьями!
